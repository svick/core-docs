# Compiler Error CS0619
'member' is obsolete: 'text'  
  
 A class member was marked with the [Obsolete]((../../csharp/programming-guide/concepts/attributes/common-attributes.md#Obsolete) attribute, such that an error will be issued when the class member is referenced.  
  
 The following sample generates CS0619:  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
