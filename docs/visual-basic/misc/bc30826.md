# &#39;EndIf&#39; statements are no longer supported; use &#39;End If&#39; instead
The `EndIf` statement is no longer supported.  
  
 **Error ID:** BC30826  
  
## To correct this error  
  
-   Use `End If` to terminate `If` statements.
