# &#39;&lt;membername1&gt;&#39; and &#39;&lt;membername2&gt;&#39; cannot overload each other because they differ only by the types of optional parameters
Overloaded members must have different data types for one or more required parameters.  
  
 **Error ID:** BC30696  
  
## To correct this error  
  
-   Define at least one required parameter as a different data type.
